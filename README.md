# CCNA Home Lab — Cisco Packet Tracer

A hands-on network lab built to demonstrate CCNA-level skills using Cisco Packet Tracer.
All topologies are fully functional with verified end-to-end connectivity.

## Lab Phases

| Phase | Topic | Status |
|-------|-------|--------|
| 1 | Environment Setup | ✅ Complete |
| 2 | Subnetting & Fundamentals | 🔄 In Progress |
| 3 | Switching & VLANs | ⏳ Upcoming |
| 4 | Routing (OSPF) | ⏳ Upcoming |
| 5 | Services (DHCP, NAT, ACLs) | ⏳ Upcoming |
| 6 | Capstone Enterprise Lab | ⏳ Upcoming |

## Technologies Demonstrated

- IPv4 addressing & VLSM subnetting
- VLANs, 802.1Q trunking, Router-on-a-Stick
- OSPFv2 (single and multi-area)
- DHCP with relay agent (ip helper-address)
- NAT/PAT (overload)
- Standard & extended ACLs
- Port security, STP, EtherChannel
- SSH hardening & device hardening

## How to Open the Lab Files

1. Install [Cisco Packet Tracer](https://www.netacad.com/courses/packet-tracer) (free via NetAcad)
2. Download the `.pkt` file from the `packet-tracer-files/` folder
3. Open in Packet Tracer and explore
